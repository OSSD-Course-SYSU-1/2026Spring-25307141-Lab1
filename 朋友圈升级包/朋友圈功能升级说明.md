### 2. 评论功能

**功能特性**:

- ✅ 评论输入框固定在底部
- ✅ 支持输入评论内容
- ✅ 提交评论添加到朋友圈
- ✅ 取消评论关闭输入框
- ✅ 评论成功后自动更新列表

**技术实现**:
```typescript
// 评论相关状态
@State private commentingMomentId: string = ''  // 正在评论的朋友圈ID
@State private commentInput: string = ''        // 评论输入内容
@State private showCommentInput: boolean = false // 是否显示评论输入框

// 开始评论
private startComment(momentId: string) {
  this.commentingMomentId = momentId
  this.commentInput = ''
  this.showCommentInput = true
  console.log(`朋友圈：开始评论朋友圈 ${momentId}`)
}

// 提交评论
private submitComment() {
  if (!this.commentingMomentId || !this.commentInput.trim()) {
    console.log("朋友圈：评论内容为空，不提交")
    return
  }
  
  // 查找对应的朋友圈
  const momentIndex = this.momentList.findIndex(item => item.id === this.commentingMomentId)
  if (momentIndex === -1) {
    console.log(`朋友圈：未找到朋友圈 ${this.commentingMomentId}`)
    return
  }
  
  // 创建评论数据
  const newComment: CommentData = {
    userName: "我",
    content: this.commentInput.trim()
  }
  
  // 添加到对应朋友圈的评论列表
  const updatedMomentList = [...this.momentList]
  updatedMomentList[momentIndex].comments = [...updatedMomentList[momentIndex].comments, newComment]
  this.momentList = updatedMomentList
  
  // 重置评论状态
  this.commentInput = ''
  this.showCommentInput = false
  this.commentingMomentId = ''
  
  console.log(`已发表评论到朋友圈 ${this.commentingMomentId}: ${this.commentInput}`)
}

// 取消评论
private cancelComment() {
  this.commentInput = ''
  this.showCommentInput = false
  this.commentingMomentId = ''
  console.log("朋友圈：取消评论")
}
```

### 3. 布局结构优化

**修改文件**: `entry/src/main/ets/pages/moment/MomentPage.ets`

**布局结构**:
```typescript
build() {
  Stack({ alignContent: Alignment.Bottom }) {
    Column() {
      // 顶部工具栏
      WechatToolbar({ title: "朋友圈" })
        .backgroundColor('#FFFFFF')
      
      // 朋友圈内容区域
      Scroll() {
        Column() {
          // 风景图区域
          Stack() { ... }
          
          // 朋友圈列表
          ForEach(this.momentList, (item: MomentItemData) => {
            MomentItem({ 
              data: item,
              onCommentClick: (momentId: string): void => this.startComment(momentId)
            })
          })
        }
      }
      .onScroll((xOffset: number, yOffset: number) => {
        this.scrollOffset = yOffset
        // 滚动事件处理
      })
    }
    
    // 评论输入框（底部固定）
    if (this.showCommentInput) {
      Column() { ... }
    }
  }
}
```

**关键技术点**:
- 使用`Stack({ alignContent: Alignment.Bottom })`实现底部固定布局
- 评论输入框覆盖在内容上方，不影响滚动
- 使用条件渲染`if (this.showCommentInput)`控制评论输入框显示

## 📱 使用说明

### 风景图功能
1. 打开朋友圈页面，顶部自动显示随机风景图
2. 风景图覆盖整个上半部分区域（300px高度）
3. 向下滚动朋友圈列表时，风景图向上移动并逐渐透明
4. 滚动400px后，风景图完全透明消失
5. 点击风景图，随机切换到另一张风景图
6. 下拉刷新时，风景图会重新随机选择

### 评论功能
1. 找到要评论的朋友圈项
2. 点击评论按钮（表情图标）
3. 底部弹出评论输入框
4. 输入评论内容
5. 点击"发送"提交评论
6. 评论成功后，评论会添加到对应朋友圈的评论列表中
7. 点击"取消"可关闭评论输入框
