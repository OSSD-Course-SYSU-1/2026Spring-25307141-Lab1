# 如何测试聊天功能

## 🎯 快速测试步骤

### 方法一：通过测试按钮（推荐）

1. **启动应用**
   - 运行HarmonyOS应用
   - 进入微信首页

2. **点击测试按钮**
   - 在微信首页顶部，你会看到一个红色的按钮：**"🔧 测试聊天功能"**
   - 点击这个按钮

3. **进入调试页面**
   - 你会看到"聊天功能调试页面"
   - 有三个测试按钮：
     - **测试1：跳转到聊天页面** - 直接进入聊天界面
     - **测试2：跳转到好友列表** - 查看好友列表
     - **测试3：返回主页** - 返回微信首页

4. **测试聊天功能**
   - 点击"测试1：跳转到聊天页面"
   - 进入聊天界面后：
     - 在底部输入框输入消息
     - 点击"发送"按钮
     - 等待1-2秒，查看自动回复

### 方法二：通过聊天列表

1. **启动应用**
   - 运行HarmonyOS应用
   - 进入微信首页

2. **点击聊天项**
   - 在微信首页，你会看到聊天列表：
     - 老婆
     - 游戏好友
     - 同事小李
     - HR小孙
     - 客户小潘
     - 老弟
   - **点击任意聊天项**，即可进入聊天页面

3. **发送消息**
   - 在聊天页面底部输入框输入消息
   - 点击"发送"按钮或按回车键
   - 消息显示在右侧绿色气泡中

4. **接收回复**
   - 发送消息后，好友会在1-2秒内自动回复
   - 回复消息显示在左侧灰色气泡中

## 🔍 如果功能不工作，请检查：

### 1. 检查应用是否成功运行
- 确保应用已成功编译
- 确保应用已在设备上运行
- 查看控制台是否有错误信息

### 2. 检查页面是否正确显示
- 微信首页是否显示聊天列表
- 是否能看到红色的测试按钮
- 点击按钮是否有反应

### 3. 检查路由跳转
- 点击聊天项后是否跳转到聊天页面
- 聊天页面是否正确显示
- 输入框和发送按钮是否可用

### 4. 检查消息发送
- 输入消息后点击发送是否有反应
- 消息是否显示在右侧气泡
- 是否收到自动回复

## 📱 功能说明

### 聊天页面功能：
- ✅ 显示好友名称
- ✅ 显示消息列表（左右气泡布局）
- ✅ 底部输入框
- ✅ 发送按钮
- ✅ 自动滚动到最新消息
- ✅ 自动回复功能

### 消息显示：
- **我的消息**：右侧绿色气泡
- **好友消息**：左侧灰色气泡

### 自动回复：
- 发送消息后1-2秒自动回复
- 回复内容：已收到、好的、在的、知道了、没问题

2. **控制台**的错误信息
3. **编译输出**的错误提示

---

**重要提示**：
- 红色测试按钮是最直接的测试方式
- 如果看不到测试按钮，说明页面没有正确加载
- 如果点击按钮没有反应，说明路由配置有问题
- 如果进入聊天页面但无法发送消息，说明聊天功能有问题
