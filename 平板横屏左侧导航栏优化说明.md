# 平板横屏左侧导航栏优化说明

## 问题描述

在平板横屏模式下，底部导航栏（微信、通讯录、发现、我）应该显示在屏幕最左侧，并且四个选项应该均匀分布在左侧栏中，没有重叠覆盖。

---

## 解决方案

### 1. 布局结构调整

**修改前：**
```typescript
Column() {
  ForEach([0, 1, 2, 3], (index: number) => {
    this.createTabletTabItem(index);
  })
}
.layoutWeight(1)
.width('100%')
.justifyContent(FlexAlign.Start)  // 从顶部开始排列
.padding({ top: 10 })
```

**修改后：**
```typescript
Column() {
  ForEach([0, 1, 2, 3], (index: number) => {
    this.createTabletTabItem(index);
  })
}
.layoutWeight(1)
.width('100%')
.justifyContent(FlexAlign.SpaceEvenly)  // 均匀分布
.padding({ top: 20, bottom: 20 })  // 上下留出边距
```

### 2. Tab项布局优化

**修改前：**
```typescript
Row() {
  Badge({ ... }) {
    Column() {
      Image(...)
      Text(...)
    }
  }
}
.width('100%')
.height(80)  // 固定高度
.justifyContent(FlexAlign.Center)
```

**修改后：**
```typescript
Column() {
  Badge({ ... }) {
    Column() {
      Image(...)
        .width(this.getIconWidth() * 1.3)  // 图标放大
        .height(this.getIconHeight() * 1.3)
      Text(...)
    }
  }
}
.layoutWeight(1)  // 平均分配空间
.width('100%')
.justifyContent(FlexAlign.Center)  // 内容居中
.borderRadius(8)  // 添加圆角
.margin({ left: 8, right: 8 })  // 左右边距
```

## 实现效果

### ✅ 均匀分布
- 四个Tab选项（微信、通讯录、发现、我）在左侧栏中均匀分布
- 使用 `justifyContent(FlexAlign.SpaceEvenly)` 确保等间距

### ✅ 自适应高度
- 每个Tab项使用 `layoutWeight(1)` 平均分配垂直空间
- 自动适应不同屏幕高度

### ✅ 无重叠覆盖
- 每个Tab项有独立的点击区域
- 添加左右边距避免视觉重叠

### ✅ 视觉优化
- 添加圆角（borderRadius: 8）
- 选中状态有背景色变化
- 图标和文字大小适配平板

## 关键技术点

### 1. FlexAlign.SpaceEvenly
- 子元素在主轴方向均匀分布
- 相邻元素之间的间距、首元素与容器起始边距、末元素与容器结束边距都相等

### 2. layoutWeight(1)
- 子元素按权重分配父容器剩余空间
- 四个Tab项权重相同，平均分配垂直空间

### 3. 响应式适配
- 图标大小：`getIconWidth() * 1.3`
- 字体大小：`getTabFontSize() * 1.1`
- 左侧栏宽度：屏幕宽度的15%（最小100，最大150）
